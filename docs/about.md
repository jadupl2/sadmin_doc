
# About SADMIN

## Description

SADMIN is working on most Linux distributions : Fedora,Debian,Ubuntu,CentOS,Red Hat,Rocky,Raspberry Pi OS.

- It's a framework to help you develop and monitor your scripts.
- A collection of scripts to automatically gather information and statistics about your Linux server farm.
- Produce performance graphic up to 18 months.
- Backup your important directories on NFS/FTP/SFTP server.
- Provide Web interface to view servers info, performance statistics, scripts results and logs.
- Manage and schedule the production of server image backup of your system using 'Rear' Open Source software.
- Include a frontend tool (sadm) to create/increase/delete filesystem (ext3,ext4,xfs) on LVM

If your are an impatient user, you may want to read the <a href="quickstart/">Quick start page</a>.
